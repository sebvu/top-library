My official submission to [Project: Library](https://www.theodinproject.com/lessons/node-path-javascript-library) of [The Odin Project](https://www.theodinproject.com/dashboard). This is a pure demonstration of frontend skill at this point of TOP with **no AI usage** and **vanilla HTML, CSS and JS.**

## Features
- Dark & light theme
- Pretty neato looking cards
- I guess a neato looking form
- Saved books and theme per session

## Methodologies used
- [Jester's Framework](https://github.com/sebvu/jester-css-framework) (my vanilla framework)
- BEM naming convention
- Josh Comeau [Modern CSS Reset](https://www.joshwcomeau.com/css/custom-css-reset/) (not a methodology but love his reset)
